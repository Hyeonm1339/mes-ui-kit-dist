# @hyeonm1339/mes-ui-kit

MES UI Kit — 공통 컴포넌트 라이브러리

## 설치

```bash
npm install @hyeonm1339/mes-ui-kit
```

## 설정

프로젝트 루트에 `.npmrc` 파일 생성:

```
@hyeonm1339:registry=https://npm.pkg.github.com
//npm.pkg.github.com/:_authToken=YOUR_GITHUB_TOKEN
```

## 사용법

```tsx
import { AppInput, AppButton, AppTable } from '@hyeonm1339/mes-ui-kit'
import '@hyeonm1339/mes-ui-kit/lib/style.css'
```
